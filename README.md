# AI编程教育套件

一个面向编程小白的综合性AI编程教育平台，提供系统化的编程入门教程、AI数据清洗功能和AI代码生成工具。

## 项目概述

本项目包含三个核心模块：

### 1. 编程入门文档模块 (programming-guide)
面向编程小白的系统化编程入门教程，采用图文结合的方式，内容涵盖：
- 基础语法教程
- 开发环境搭建指南
- 代码编写规范
- 常见问题解决方案
- 实践案例和示例代码

### 2. AI数据清洗模块 (ai-data-cleaning)
基于AI的自动编码数据清洗功能，能够：
- 自动识别数据中的异常值、缺失值和不一致数据
- 提供智能化清洗解决方案
- 支持自定义清洗规则配置
- 处理多种数据格式

### 3. AI课题代码生成模块 (ai-code-generator)
AI辅助课题代码生成系统，支持：
- 根据课题需求描述自动生成代码框架
- 支持多种编程语言
- 提供详细的代码注释和使用说明
- 确保代码的可维护性和可扩展性

## 项目结构

```
AI-Programming-Education-Suite/
├── programming-guide/          # 编程入门文档模块
│   ├── docs/                   # 文档资料
│   ├── assets/                 # 图片、视频等资源
│   ├── tutorials/              # 教程内容
│   └── examples/               # 示例代码
├── ai-data-cleaning/           # AI数据清洗模块
│   ├── src/                    # 源代码
│   ├── models/                 # AI模型文件
│   ├── config/                 # 配置文件
│   ├── tests/                  # 测试文件
│   ├── examples/               # 使用示例
│   └── docs/                   # 模块文档
└── ai-code-generator/          # AI课题代码生成模块
    ├── src/                    # 源代码
    ├── templates/              # 代码模板
    ├── models/                 # AI模型文件
    ├── config/                 # 配置文件
    ├── tests/                  # 测试文件
    ├── examples/               # 使用示例
    └── docs/                   # 模块文档
```

## 技术栈

- 前端：HTML, CSS, JavaScript
- 后端：Python, Node.js
- AI/ML：TensorFlow, PyTorch, Scikit-learn
- 数据处理：Pandas, NumPy
- 文档：Markdown, MkDocs
